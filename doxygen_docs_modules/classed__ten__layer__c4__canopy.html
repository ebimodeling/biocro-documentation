Detailed Description

A child class of multilayer_canopy_photosynthesis where:

  • The canopy module is set to the ed_ten_layer_canopy_properties module
  • The leaf module is set to the ed_c4_leaf_photosynthesis4 module
  • The number of layers is set to 10 Instances of this class can be created using the module factory.
