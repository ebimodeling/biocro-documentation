Detailed Description

template<typename T>
class module_creator_impl< T >

A template class that provides a common interface to the constructor and essential static methods of a concrete module class.

Here, the typename T should be the type of a concrete module class; in other words, T should be a class derived from module. The module_creator_impl class enforces the requirement that each concrete module should have static methods get_inputs(), get_outputs(), and get_name() that return strings or vectors of strings.
