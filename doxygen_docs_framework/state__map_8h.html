◆ state_vector_map

using state_vector_map = std::unordered_map<std::string, std::vector<double> >

A state_vector_map represents a chronologically-ordered sequence of states of a system over some period of time.

The keys of the map are quantities of the system; each value is a vector representing the sequence of values of one of those quantities as it changes over time. To be a valid representation, each of the mapped-to vectors should be of the same length.
