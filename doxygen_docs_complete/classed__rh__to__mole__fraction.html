Detailed Description

Just converts relative humidity values from weather data into atmospheric mole fractions of water vapor. Currently only intended for use by Ed.

Here we just use the definitions of mole fraction and relative humidity:

mole_fraction = partial_pressure / atmospheric_pressure = saturation_water_vapor_pressure * relative_humidity / atmospheric_pressure
