BioCro C++ Framework VERSION 1.1.0

  • Two new values were added to the physical_constants namespace (defined in constants.h: dr_stomata and dr_boundary.
  • Several new functions for finding the roots of quadratic equations were added (see quadratic_root.h and quadratic_root.cpp).
  • Both overloaded versions of dynamical_system::update_drivers were updated to prevent out-of-bounds errors when accessing vector elements, an important bug fix.
